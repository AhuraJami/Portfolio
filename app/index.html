Recent Project

Co-Simulation Framework

© Networked Systems Lab

Built from scratch and powered by Unreal Engine, Co-Sim is a comprehensive simulation framework designed specifically for vehicular active safety system analysis. It comprises different domains of communication, transportation, and vehicle safety systems. In addition, it incorporates a driver reaction model and a hazard detection package that rely on situational data collected by local sensors or via communication. Using this tool, we can study and analyze the effect of entailed components on performance of the safety system in a large scale traffic scenario.
